# Getting Started with Quartz

This guide will help you understand how to work with this Quartz-based digital garden.

## Writing Content

All content is written in Markdown and placed in the `/content` directory. The folder structure in `/content` mirrors the URL structure of the site.

### Frontmatter

Each page can have frontmatter at the top:

```yaml
---
title: Page Title
tags:
  - tag1
  - tag2
published: true
---
```

## Features to Explore

### Wiki-links

You can link to other pages using Wiki-style links: `[[other-page]]`

### Search

Use the search bar in the sidebar to quickly find content across all pages.

### Graph View

The graph visualization shows connections between pages. Click on nodes to navigate.

### Table of Contents

On longer pages, the table of contents appears on the right sidebar for easy navigation.
